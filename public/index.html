Banking Features

  • Account Management - Create and manage accounts
  • Local Transfers - Transfer money between your accounts
  • Interbank Transfers - Send money to other banks via Central Bank
  • Secure Authentication - JWT-based authentication system

Developer Tools

Note: These tools are for development purposes only.

  • Log Viewer - View and analyze application logs
  • Test Registration - Test user registration
  • View Users - List all registered users

API Endpoints

Here are some of the main API endpoints:

  • /api/users/register - Register a new user
  • /api/users/login - User login
  • /api/accounts - Manage accounts
  • /api/transactions - Make transactions
